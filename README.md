âœ¨ Fitur Utama (Key Features)
Aplikasi ini dilengkapi dengan serangkaian fitur untuk memfasilitasi seluruh proses pemesanan:

- Splash Screen:
Tampilan awal yang menarik untuk menunjukkan logo dan identitas aplikasi saat loading.

- Search Flight Page:
Halaman Utama tempat pengguna dapat mencari penerbangan dengan mengisi detail seperti rute, tanggal, dan jumlah penumpang.
Menyediakan form pencarian yang intuitif.

- Search Result Page:
Menampilkan list hasil pencarian berdasarkan kriteria yang dimasukkan.
Setiap hasil menampilkan informasi kunci: Nama Maskapai, Jam Berangkat, Jam Tiba, dan Harga tiket.

- Booking Detail Page:
Halaman form untuk melengkapi pembelian.
Memungkinkan pengguna mengisi Data Pemesan dan Data Penumpang.
Data yang dikumpulkan meliputi: Nama, Nomor KTP, Email, dan Nomor HP.

- My Tickets Page:
Halaman personal yang menampilkan list tiket yang sudah dibeli.
Tiket disajikan dalam format Card yang mudah dibaca, berisi Kode Booking dan detail Rute Penerbangan.
